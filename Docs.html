빠른 이동

아래 버튼을 누르면 해당 섹션으로 바로 이동합니다.

1) 시작하기

탭 이동

상단 탭으로 기능을 전환합니다.

테마 전환

달/해 아이콘으로 다크/라이트 모드 전환.

설명서 열기

우측 상단 설명서 버튼으로 즉시 열기.

중요. 변경사항이 안 보이면 먼저 새로고침을 해주세요.

2) 1분 빠른 사용 순서
  1. 작성 탭으로 이동
  2. 작성자 입력
  3. 구분 선택 (기자재/운영진 확인/인수인계)
  4. 이슈 내용 작성 + 필요하면 글자 꾸미기
  5. 필요 시 사진 첨부
  6. 근무일지 전송
  7. 조회 탭에서 방금 작성한 건 확인
3) 공지사항 탭 사용법
  • 목록에서 공지를 클릭하면 상세 모달이 열립니다.
  • 상세 모달에서 제목, 본문, 이미지를 확인합니다.
  • 중요 공지는 더 눈에 띄는 스타일로 강조됩니다.
  • 본문에 적용된 글자 크기/굵기/색상도 반영되어 보입니다.

팁. 공지 내용이 길면 모달 내부에서 스크롤해 전체 내용을 읽을 수 있습니다.

4) 작성 탭 상세 가이드

필수 입력

  • 작성자
  • 이슈 내용
  • 구분이 기자재면 주기번호

선택 입력

  • 현장 사진
  • 추가 주기번호
  • 문서 꾸미기(색/링크/이모지)
  1. 발생 일자는 달력에서 선택합니다.
  2. 작성자를 입력합니다.
  3. 구분을 고릅니다.
  4. 기자재 이슈면 주기번호 훔쳐보기로 검색해 정확히 입력합니다.
  5. 이슈 내용을 문장 단위로 명확히 씁니다. (무엇이, 어디서, 어떻게 문제인지)
  6. 필요 시 사진을 업로드합니다.
  7. 하단 근무일지 전송 버튼을 눌러 저장합니다.

참고. 현장 사진은 작성 영역의 사진 업로드(이미지 추가) 버튼에서 첨부할 수 있습니다.

5) 이슈내용 꾸미기 도구 (리치 에디터)
굵게 / 기울임 / 취소선
글자 크게 / 작게 (단계별)
글자 색상 + 세부색상
하이퍼링크 추가/해제
이모지 입력 😀 ✅ 🚨
활성화된 버튼은 더 진하게 표시

사용 방법. 먼저 텍스트를 드래그 선택한 뒤 버튼을 누르면 해당 구간에만 적용됩니다.

단축키. Windows : Ctrl+B / Ctrl+I | Mac : Command+B / Command+I

이모지. Mac : Control + Command + Space, Windows : Win + . 로 이모지/기호 창을 열 수 있습니다.

6) 조회 탭 사용법
  1. 상단의 구분 버튼(기자재/운영진 확인/인수인계)을 선택합니다.
  2. 필요하면 필터 버튼으로 품목/날짜/조치결과를 조합합니다.
  3. 리스트에서 항목을 클릭하면 상세 모달이 열립니다.
  4. 상세 모달에서 이슈내용, 이미지, 조치 상태를 확인합니다.

작성 시 적용한 글자 스타일(굵기/크기/색/링크)은 조회 상세에서도 유지됩니다.

7) 교구 확인 탭 사용법
  • 품목별 카드(아코디언)를 클릭해 세부 목록을 펼칩니다.
  • 주기번호, 보관 위치, 대여 정보, 파손 상태를 확인합니다.
  • 상단 새로고침 버튼으로 최신 정보를 다시 불러옵니다.
8) 자주 하는 실수와 해결법

전송 버튼이 비활성화

작성자/이슈내용/필수 입력값이 빠지면 전송할 수 없습니다. 입력 후 다시 확인하세요.

링크가 안 걸리는 것처럼 보임

텍스트를 선택한 뒤 링크 버튼을 누르세요. 선택이 없으면 커서 위치에 URL 텍스트 링크가 삽입됩니다.

이미지가 보이지 않음

원본 이미지 URL 또는 접근 권한 문제일 수 있습니다. 파일 공개 범위를 확인하세요.

다크모드에서 글자가 너무 어두움

검정색 고정 글자 대신 기본색 또는 밝은 계열 색상을 사용하면 가독성이 좋아집니다.

9) 자주 묻는 질문 (FAQ)

Q1. 작성한 글자 크기/색상도 조회에서 그대로 보이나요?
A. 네. 작성 시 적용한 스타일은 저장되어 조회/상세에서도 반영됩니다.

Q2. 공지사항도 글자 스타일이 반영되나요?
A. 네. 공지사항 상세에서도 글자 크기/강조가 반영됩니다.
